Веб-приложение "Агенство недвижимости"

* Бэкенд: Django REST Framework
* Фронтенд: React, Redux Toolkit, Sass, Reactstrap
* База данных: PostgreSQL
* Сервер статики: Minio
* Другие используемые технологии: Docker, Nginx
