Веб-приложение "Агенство недвижимости"

* Бэкенд: Django REST Framework
* Фронтенд: React, Redux Toolkit
* База данных: PostgreSQL
* Сервер статики: Minio
* Другие используемые технологии: Docker, Nginx, Sass, Reactstrap
